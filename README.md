- Instant website analysis with AI-powered insights
- Metrics for SEO, Performance, Mobile Responsiveness, and Lead Generation
- Actionable recommendations for improvement

### Interactive Demos

- **Content Generator**: AI-powered content creation for roofing businesses

### Core Services
- AI Chatbot Integration

- Website Design & Development
- Social Media Management
- PPC Advertising
- Content Marketing

## Usage

Simply open `index.html` in a web browser to view the landing page. No build process or dependencies required.

For local development with a server:
```bash
python3 -m http.server 8000
# Then open http://localhost:8000
```
